
# 🎬 CineVerse — Movie Recommendation System

A smart and interactive movie recommendation web app.

## Features
- 🔍 Search movies and get instant recommendations
- 🏆 Top 10 Movies with posters
- 🎭 Browse by Category
- ⭐ Rate movies (login required)
- 👤 Login & Signup system
- 🔄 Personalized recommendations based on your ratings
